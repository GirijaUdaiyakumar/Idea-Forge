# 🚀 Idea Forge — AI Idea Generator

## 🧠 SPEC (What this project does)

Idea Forge is a full-stack web application that transforms user input ideas into structured AI-style project suggestions.

Users enter an idea like:
> "Smart classroom"

The system generates:
- Project Title
- Description
- Key Features
- Tech Stack

This demonstrates a working full-stack scaffold using React + Node.js + Express.

---

## 🎯 CORE SCREENS

### 1. Home Screen
- Input field for idea
- "Generate Ideas" button

### 2. Result Screen
- AI-generated project title
- Description
- Features list
- Tech stack list

---

## 🛠 TECH STACK

- Frontend: React + Vite
- Backend: Node.js + Express
- API: REST API (JSON response)
